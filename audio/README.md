# 🔊 Аудио жүйесі (жаңартылған жоспар)

Web Speech API көптеген әріптер мен сөздерді қазақша оқи алады, бірақ
**қазақтың 8 арнайы әрпін** дұрыс айта алмайды. Сондықтан тек сол 8 әріп
үшін нағыз дауыс (MP3) қолданылады, қалғаны автоматты Web Speech-пен айтылады.

## Не істеу керек

ElevenLabs-те (немесе кез келген қазақша TTS) мына **8 әріпті** жазып,
`audio/letters/` папкасына салыңыз — толық нұсқаулық сонда:
**[audio/letters/README.md](letters/README.md)**

| Файл | Дыбыс |
|------|-------|
| `audio/letters/а.mp3` | а |
| `audio/letters/ә.mp3` | ә |
| `audio/letters/ғ.mp3` | ғ |
| `audio/letters/қ.mp3` | қ |
| `audio/letters/ң.mp3` | ң |
| `audio/letters/ө.mp3` | ө |
| `audio/letters/ұ.mp3` | ұ |
| `audio/letters/ү.mp3` | ү |

## Қалай жұмыс істейді

- **Әріп дыбысы** (`playLetter`): 8 арнайы әріп → MP3, басқасы → Web Speech.
- **Сөз дыбысы** (`playWord`): сурет/сөзді толық айту → Web Speech.
- **Кері байланыс** (`playFeedback`): «Дұрыс! Жарайсың!» / «Қайталап көр».
- **Дыбыс эффекттері** (`sfx`): дұрыс/қате/марапат үндері (Web Audio, файлсыз).

> MP3 болмаса — бәрі Web Speech-пен жұмыс істейді (ешнәрсе бұзылмайды).
> 8 файлды қосқан соң — нағыз қазақша дауыс автоматты қосылады.
