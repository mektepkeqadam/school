# 🔊 Қазақша арнайы әріптердің дыбыстары

Осы папкаға **8 MP3 файл** салыңыз (ElevenLabs-те қазақша дауыспен жазылған).
Файл атаулары — **кіші әріппен**, дәл осылай:

| Файл | Әріп (дыбыс) |
|------|--------------|
| `а.mp3` | а |
| `ә.mp3` | ә |
| `ғ.mp3` | ғ |
| `қ.mp3` | қ |
| `ң.mp3` | ң |
| `ө.mp3` | ө |
| `ұ.mp3` | ұ |
| `ү.mp3` | ү |

Қалған әріптер мен сөздер Web Speech API арқылы автоматты айтылады — оларға файл қажет емес.

> Файл жоқ болса, қолданба сол әріпті Web Speech-пен айтады (ешнәрсе бұзылмайды).
> Файлды қосқан соң — нағыз қазақша дауыс автоматты қосылады.
